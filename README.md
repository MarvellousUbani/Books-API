### MyReads Project
> Features: search for books, move books around categories, add new / remove old books

A simple book search app built with react

## Built With
- React
- Javascript
- HTML/CSS(Flexbox)

## How to Run the App Locally
```

   cd books-api
   npm install
   npm start

```
